This project is a command-line based Hangman game developed in Python, designed to demonstrate core programming concepts such as control flow, data structures, and user interaction. The game randomly selects a word from a predefined list, and the player must guess the word one letter at a time within a limited number of attempts. Each incorrect guess brings the player closer to losing, visually represented through the traditional Hangman stages.

The implementation focuses on clean logic and modular design. Python data structures like lists and sets are used to efficiently track guessed letters, remaining attempts, and the current state of the word. The game dynamically updates the display after every guess, showing correctly guessed letters in their respective positions while masking the remaining ones. Input validation ensures that only valid single-letter guesses are accepted, preventing crashes and improving user experience.

The project also emphasizes user feedback and interaction. Clear prompts guide the player through the game, while informative messages indicate whether a guess is correct, repeated, or invalid. At the end of each round, the game displays whether the player has won or lost and reveals the correct word, enhancing engagement.
